Systmctl - Journal/Tutorial: Installing Kali Linux | Pentesting

Date: 10/06/2025
Difficulty: Beginner

Hello! I'm Systmctl. This is the first documentation I am making in the Journal of Knowledge. This is beginner-friendly, and you can do this.

To begin, you need to download a virtualizer (Virtual Machine) in order to run the system within a "virtual" hard drive. There are several options available, and the best ones will be listed below.

  • Q-Emu
  • Oracle Machine
  • VmWare

As you have probably never done this before, I recommend using Oracle Machine, as it is the most accessible and easiest option to use.

Once you access the website, you will see an option labeled "Windows Hosts", click on it. This is the version intended for Windows operating systems.

Download the .exe file, install it properly, and open the installed program.

Now that we have installed our virtualizer from Oracle, we need to install the .iso image of Kali Linux. In this case, you can install any operating system to practice hacking in general; however, Linux distributions offer greater ease when handling tools, are lighter systems, and can be configured to become true security fortresses.

You also don't need to be attached to Kali Linux — it is essentially just a Linux system with pre-installed tools. You could install Parrot OS, which is also focused on hacking, or even choose any other Linux distro and manually install the tools from scratch. However, since you have probably never done this before, it is recommended that you follow my step-by-step guide exactly.

I recommend installing Q-Bittorrent, a torrent client based on the Bittorrent protocol. It is an open-source and trustworthy alternative to uTorrent. In the hacking world, always prefer open-source software.

After that, go to the Kali Linux website and download the ISO image using Qbittorrent, then wait for the download to complete.

After downloading the Kali Linux .iso file, open the Oracle VirtualBox software and click on "New".

After clicking on "New", assign a name to your Virtual Machine and select the kali-linux.iso image from the directory where you downloaded it.

After that, click on 'Hardware' and configure the processing and RAM settings for the virtual machine.

In this case, you will notice that there are these bars, and up to a certain point they remain green. This depends heavily on your hardware overall; however, adjust the settings so that they stay within the green area, exactly as shown in my example:

After completing that, click on 'Hard Disk'.

This option depends on the available space on your computer’s hard drive, specifically your HDD or SSD. Select an amount based on your available storage. In my case, I will choose 60GB.

Once the configuration is complete, finalize the setup.

With the virtual machine configured and finalized, let’s click on Start.

After clicking Start, if you have followed the steps correctly, this software should appear:

It is extremely important that you select Graphical Install, as it will use a graphical interface to assist you with the installation.

There are other options, such as Normal Install, which basically requires you to use only the terminal to configure the system from scratch. This option is intended for advanced users and will not be covered in this documentation. However, if you want to challenge yourself and gain extensive knowledge, you may choose Normal Install, but it is not recommended for beginners.

After selecting Graphical Install, if everything proceeds correctly, you will notice that you have entered the manual graphical installation environment.

On this first screen, you should select the language for your Operating System and the installation interface.

On the second screen, you should select your location.

On the third screen, select your keyboard layout correctly.

The installer will perform some basic configurations, just wait patiently.

On this screen, enter your Hostname, the name your computer will use on the network. In my case, I prefer to use "desktop" to maintain my OPSEC and blend in with others.

Here, you will choose the password for your User account as well as for the Root user.

On this screen, it is very important that you select the first option.

On this screen, you must also select the first option without exception.

Here, you should select the "yes" option to allow modifications to the virtual hard disk.

The installer will then proceed to install essential components, including the base system and other necessary files. This step may take some time depending on your computer or laptop’s hardware specifications.

Here, leave the settings as they are, do not add or remove anything.

This part takes the longest, and it is normal for it to take quite some time because it is installing the Desktop Environment and the pre-installed tools.

After the software installation is complete, on this screen you should select "yes".

After installing the bootloader, you will complete the installation.

Then, enter the username you chose along with the corresponding password.
